About This Project

Problem Statement


Due to technology innovation, one of the most significant problems regarding vaccination misinformation and fake news is being generated and disseminated at an alarming rate. Misinformation regarding vaccinations, which has the potential to create vaccine hesitancy in certain individuals, is the second issue that must be addressed.

Objective


The objectives of this project are outlined as follows:

1. To detect and classify the misinformation about vaccines based on the Twitter dataset.
2. To build a web-based system that can classify and display the result using the classification machine learning technique.
3. To test the functionality and accuracy of the system.

Aim


To aim to develop a web-based system that can detect and classify vaccine misinformation by implementing a classification method using machine learning based on the collected Twitter dataset to provide more information about vaccination misinformation.
